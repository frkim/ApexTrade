# ApexTrade

A trading bot designed to monitor and trade stocks, currencies, and cryptocurrencies.

## Overview

ApexTrade is an automated trading system that monitors one or multiple financial instruments and executes buy/sell actions based on market conditions.

### Features

- Monitor stocks, currencies, and cryptocurrencies
- Automated buy and sell actions
- Multi-asset support
